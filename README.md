## Getting Started
  1. Download or clone this repository with all the code available.
  2. Next open VS code and open the repository in VS code.
  3. Next make sure to have all the extentions listed in the section titled "Extentions Needed To Run"
  4. Open any HTML page provided in the repository (Generator.html, Index.html, or Analyzer.html)
  5. In the top right corner there will be a button called open preview
